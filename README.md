# Expression Mission

A browser-based algebra game for Form 1 students at Caribbean Union College Secondary School.

Covers three levels of early algebraic thinking:

- **Level 1 — Expansion:** Distribute the outer term across both terms inside the bracket
- **Level 2 — Substitution:** Replace x with a given value and evaluate the expression  
- **Level 3 — Does It Work?** Test whether a given value makes an equation true

## How to play

Open the link below on any phone or computer browser. No installation, no login, no internet required after the first load.

**Play here:** https://legallev-teach.github.io/expression-mission/

## For students

- Enter your name to begin
- Complete each level in order
- Screenshot your **Flight Record** at the end and send it to your teacher

## For teachers

- Tap the 🔒 icon on the title screen
- Enter PIN to access the Teacher Dashboard
- Paste student Flight Record codes into the Bulk Decoder to see results instantly

## Technical notes

- Single HTML file — no server, no database, no dependencies
- Progress saved automatically to the device
- Works on Chrome, Safari, Firefox
- Best on mobile
